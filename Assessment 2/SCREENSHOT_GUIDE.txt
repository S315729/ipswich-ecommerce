==================
SCREENSHOT CAPTURE TIPS
==================

1. WINDOWS SNIPPING TOOL:
   - Press: Windows + Shift + S
   - Select area to capture
   - Image copied to clipboard
   - Open Paint or Paint 3D to save
   - Save as PNG in "Assessment 2\screenshots\" folder

2. BROWSER SCREENSHOTS:
   - Press F12 to open DevTools
   - Press Ctrl + Shift + P
   - Type "Capture full size screenshot"
   - Image saves to Downloads folder
   - Move to screenshots folder

3. POWERSHELL SCREENSHOTS:
   - Make PowerShell window large enough to show full output
   - Use Windows + Shift + S to capture
   - Include command and full output in capture

4. VS CODE SCREENSHOTS:
   - Open relevant file
   - Zoom to readable font size (Ctrl + Plus)
   - Capture with Windows + Shift + S
   - Include VS Code UI for context
